For this assignment, I added a few new features to the game. I modified the thisx and thisy variables to depend on the window’s width and height. This way when the bee changes position, it remains within the screen. I also created a variable pause_boolean to keep track of when the game is paused. In the showScore() function, I reference this value so the points only increment when the bee is moving. I also disabled the pause and start buttons after they are clicked. I then created toggle buttons to display this section and the instructions. Lastly, I used an IF statement in the showScore() function to change the background image and to play a sound when 10 points are reached.

Some other modifications I made were adding bootstrap buttons, and adding a transition for the bee image. I also created the two background gifs using Canva.
